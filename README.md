# Rainbow 🌈

The real extension lives in ./vscode-language-bar.

Rainbow adds a GitHub-style language bar to Visual Studio Code, showing the language composition of your workspace using the familiar colorful segments seen on GitHub repositories.

I originally made this because I work with a lot of different languages and genuinely enjoy seeing the colorful breakdown grow and change over time. It's a tiny feature, but it makes multi-language projects feel much more alive.

If you're the kind of developer who likes statistics, colorful dashboards, or simply seeing Rust fight Java for screen space, Rainbow is for you. 🌈
